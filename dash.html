    • About Game

    This game uses a simple click system to control different vehicles (or click for the Steam distribution).1 The objective of the game is to complete the levels; however, if the player makes a bad jump or move against an obstacle, they will have to start the level from the beginning. In the same way the game has the practice mode, which unlike the normal mode, checkpoints that are available in this mode can be placed, allowing the player to start in these places instead of the beginning. The video game features 21 official levels, of which eighteen are playable from the start. The lengths of the main levels in the game are approximately 1:20 to 1:45 minutes. Levels are ranked by difficulty, from easy to "demon"; in user levels with variations of "Easy Demon", "Medium Demon", "Hard Demon", "Insane Demon" and "Extreme Demon". In the game there are seven types of modes to play, which are called as vehicles and each behaves differently, such as cube, ship, ball, UFO, wave, robot and spider. In the game there are several types of portals, these allow you to change vehicles, change size, change gravity, change direction, change speed, change position (via teleportation) and duplicate the icon by making the user handle two icons at once. The pads are objects that, when in contact with the vehicle, cause it to jump (to a slightly greater distance than the orbs) automatically (with the exception of the wave, which only interacts with the pads that change gravity). Orbs are small circles that interact with the player only if they click when the icon passes their position.
